# ScienceLink - Platforma Społeczności Naukowej

Projekt zmigrowany z React+Vite do Next.js (App Router).

## Wymagania
- Node.js 18+
- npm

## Instalacja

```bash
npm install
```

*Uwaga: W przypadku problemów z zależnościami peer dependencies (szczególnie przy React 19), użyj:*
```bash
npm install --legacy-peer-deps
```

## Uruchomienie (Development)

```bash
npm run dev
```
Aplikacja dostępna pod adresem: http://localhost:3000

## Budowanie i Uruchomienie (Produkcja)

```bash
npm run build
npm start
```

## Testy

### Unit & Component Tests
```bash
npm run test
```

### E2E Tests (Playwright)
```bash
npx playwright test
```
